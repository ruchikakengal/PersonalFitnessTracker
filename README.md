## 🧠 About the Project  

Personal Fitness Tracker is a sleek and responsive web application designed to help users monitor essential health metrics like **BMI** and stay consistent with their fitness journey.  

Built using modern frontend technologies and optimized for performance, this project delivers a smooth and engaging user experience across all devices.

---

## ✨ Features  

- 📊 Track BMI and basic health metrics  
- 🎯 Clean, minimal, and intuitive UI  
- ⚡ Fast performance with optimized build  
- 📱 Fully responsive across devices  
- 🌍 Live deployed application
